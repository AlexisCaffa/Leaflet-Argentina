## Mapa de provincias argentinas con [Leaflet](http://leafletjs.com)

Para salir andando rápido:
- Con Python
```bash
python2 -m SimpleHTTPServer
python3 -m http.server
```

- Con NodeJS
```bash
 npm i http-server -g
 http-server
```

- Con PHP
```bash
php -S localhost:8000
```
